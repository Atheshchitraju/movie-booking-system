A full-stack movie ticket booking platform that allows users to browse movies, select seats, and book tickets online. The system also includes an admin panel for managing movies, theaters, and shows.


## 🚀 Features

### User
- User registration and login
- Browse available movies
- View movie details
- Select theater and show
- Seat selection
- Book tickets
- View booking history

### Admin
- Add and manage movies
- Add and manage theaters
- Add and manage shows
- View all bookings
- Manage platform data from admin dashboard
